## 4. 湖南饮食文化特征

### 口味偏好

| 口味 | 消费者占比 | 菜品设计建议 |
|------|-----------|-------------|
| 中辣 | 45% | 主力菜品定位 |
| 微辣 | 30% | 必须提供微辣选项 |
| 特辣 | 15% | 作为特色/卖点菜品 |
| 不辣 | 10% | 儿童/老人/外地客必备 |

### 用餐习惯

- **午市**：写字楼商圈午市占比 50-55%（全国偏高）
- **宵夜文化**：长沙宵夜消费全国前 3，夜间 21:00-02:00 贡献 10-15% 营收
- **包厢文化**：湖南商务宴请/朋友聚餐偏好包厢，包厢收入占正餐 30-40%
- **烟酒消费**：湖南饮酒文化强，酒水毛利 60-70%，是利润重要补充
- **嗦粉习惯**：湖南米粉消费频次极高，早餐/宵夜场景刚需

### 季节性消费

| 季节 | 消费特征 | 运营建议 |
|------|---------|---------|
| 春季(3-5月) | 小龙虾季，踏青聚餐增多 | 推小龙虾套餐，增加户外座位 |
| 夏季(6-8月) | 夜宵经济高峰，啤酒消费旺季 | 延长营业时间，推烧烤/啤酒 |
| 秋季(9-11月) | 蟹季，婚宴高峰 | 推大闸蟹，宴会营销 |
| 冬季(12-2月) | 火锅/砂锅旺季，年夜饭 | 推暖身菜品，预售年夜饭 |

## 5. 种子客户所在市场分析

### 尝在一起（5 店，湘菜正餐）

| 维度 | 分析 |
|------|------|
| 定位 | 新派湘菜，80-120元客单 |
| 客群 | 25-45岁，白领/家庭聚餐 |
| 核心痛点 | 食材损耗不透明（5 店各自采购，无统一标准）|
| 巡店效率 | 老板每周只能巡 2-3 店，剩余门店靠店长自觉 |
| 屯象价值 | 实时损耗监控 + 多店对标 → 预期节省 ¥2-3万/月 |

### 最黔线（12 店，贵州菜连锁）

| 维度 | 分析 |
|------|------|
| 定位 | 地道贵州菜，65-90元客单 |
| 客群 | 大众消费，社区/商圈混合 |
| 核心痛点 | 新店扩张期标准化难（12 店口味一致性差）|
| 管理挑战 | 店长能力参差不齐，中央厨房覆盖不足 |
| 屯象价值 | KPI 对标 + 标准化监控 → 新店爬坡期缩短 30% |

### 徐记海鲜（多店，高端海鲜）

| 维度 | 分析 |
|------|------|
| 定位 | 高端海鲜正餐，150-250元客单 |
| 客群 | 商务宴请/高端家庭 |
| 核心痛点 | 高食材成本率(38-42%)，生鲜损耗大 |
| 供应链挑战 | 海鲜保鲜要求高，损耗敏感 |
| 屯象价值 | 精准采购 + 损耗追踪 → 食材率降 2% = ¥10万+/月 |
