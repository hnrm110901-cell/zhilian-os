# 智链OS 演示后台启动指南

## 快速启动

### 终端1：启动后端API服务

```bash
# 进入API网关目录
cd /Users/lichun/Desktop/zhilian-os/apps/api-gateway

# 启动FastAPI服务
uvicorn src.main:app --reload --host 0.0.0.0 --port 8000
```

启动成功后，您可以访问：
- API文档：http://localhost:8000/docs
- 健康检查：http://localhost:8000/health

### 终端2：启动前端Web应用

```bash
# 进入Web应用目录
cd /Users/lichun/Desktop/zhilian-os/apps/web

# 启动Vite开发服务器
npm run dev
```

启动成功后，您可以访问：
- 前端应用：http://localhost:3000 (已配置端口)

## 新功能体验

### 1. 主题切换
- 点击顶部导航栏的灯泡图标切换亮色/暗色主题
- 或使用快捷键：Ctrl+Shift+T

- 点击顶部导航栏的搜索图标
- 或使用快捷键：Ctrl+K

- Ctrl+K：打开全局搜索

- 访问 http://localhost:3000/mobile 查看移动端界面
- 支持主题切换和响应式设计

## 故障排查

### 后端启动失败
1. 检查Python环境：`python --version` (需要Python 3.8+)
2. 安装依赖：`pip install -r requirements.txt`
3. 检查端口占用：`lsof -i :8000`

### 前端启动失败
1. 检查Node.js版本：`node --version` (需要Node.js 16+)
2. 安装依赖：`npm install`
3. 检查端口占用：`lsof -i :5173`

### API连接失败
检查前端API配置文件：`apps/web/src/services/config.ts`
确保API_BASE_URL指向正确的后端地址。

## 停止服务

在各自的终端窗口中按 Ctrl+C 停止服务。
