## 1. 系统简介

**智链OS** 是面向餐饮连锁的智能体操作系统，内置 10 个 AI Agent，将排班、库存、服务质量、客户运营等日常决策自动化，让管理者从繁琐的数据整合中解放出来，专注于门店现场。

### 核心价值

| 场景 | 传统方式 | 智链OS |
|------|---------|--------|
| 排班 | 手动排，2小时以上 | AI 自动生成，5分钟确认 |
| 库存预警 | 每天手动盘点 | 实时监控，提前推送告警 |
| 食材成本 | 月末对账才发现超标 | 每日差异分析，偏高立即预警 |
| 差评处理 | 人工发现，响应慢 | 48小时内自动触发修复旅程 |
| 营业数据 | 多系统手动汇总 | 每日自动生成备战板 |

### 10 个 AI Agent 一览

| Agent | 核心功能 |
|-------|---------|
| 排班 Agent | 流量预测 + 最优排班生成 |
| 订单 Agent | 预定/排队/点单/结账全流程 |
| 库存 Agent | 实时监控 + 补货预警 + 过期预警 |
| 服务质量 Agent | 客诉处理 + 满意度监控 |
| 培训 Agent | 个性化培训计划 + 进度追踪 |
| 决策 Agent | KPI 分析 + 经营建议 |
| 预定 Agent | 智能选桌 + 宴会管理 |
| 私域运营 Agent | RFM 分层 + 旅程自动化 |
| 绩效 Agent | 员工绩效评分 + 考核报告 |
| 运维 Agent | 系统健康监控 + 告警 |

## 2. 登录与角色权限

### 2.1 登录方式

**网页端（推荐）**
```
https://your-domain.com
```
支持以下三种方式登录：
- **账号密码**：用户名 + 密码
- **企业微信扫码**：推荐，免密直接进入

**企业微信端**

## 3. 企业微信自然语言交互

智链OS 接入企业微信，支持用自然语言向 AI 发送指令，无需打开网页。

### 3.1 如何发起对话

在企业微信中找到「智链OS助手」机器人，直接发送消息即可。

### 3.2 常用指令示例

**查询类（任意角色均可）**

```
今天的库存情况怎么样？
昨天的营业额是多少？
最近7天的差评有哪些？
下周的排班安排是什么？
```

**操作类（店长权限）**

```
帮我生成明天的排班方案
审批今日备战板
查看食材成本分析
触发新客激活旅程给 C001
```

**分析类（店长/管理员）**

```
分析本月食材成本异常原因
哪些员工本周绩效最高？
预测明天的客流量
```

### 3.3 指令技巧

- **加上时间范围**效果更好：「昨天」「本周」「2月份」
- **指定门店**（多店管理时）：「S001 门店本周排班」
- 如 AI 未理解，可重新描述或换一种说法

### 3.4 每日备战板推送

每天 **7:00** 系统自动向店长推送当日备战板摘要，包含：
- 昨日关键数据回顾
- 今日预测营收和注意事项
- 待审批事项提醒

## 4. 每日备战板（Daily Hub）

每日备战板是店长最重要的决策工具，每天汇聚五个模块的数据供审批确认。

### 4.1 访问入口

- **网页端**：主菜单 → 「每日备战板」
- **企业微信**：每天早上自动推送，点击「查看详情」

### 4.2 五大模块说明

#### 模块一：昨日复盘

| 字段 | 说明 |
|------|------|
| 实际营收 | 昨日实收金额（元） |
| 订单数 | 昨日订单总数 |
| 健康分 | 综合经营健康度（0-100分） |
| 亮点 | 系统识别的昨日经营亮点 |
| 告警 | 需要关注的问题（食材超标、服务异常等） |
| 食材成本 | 实际成本率 vs 理论成本率（差异 >2% 警告，>5% 严重告警） |

> **食材成本告警说明**
> - 🟡 黄色：成本率偏高（差异 2-5%），注意观察
> - 🔴 红色：成本率严重超标（差异 ≥5% 或实际成本率 ≥35%），需立即排查

#### 模块二：外部因子

系统自动获取明日的天气、节假日、特殊日子等外部信息，影响系数越高代表预期客流越大。

#### 模块三：明日预测

- **宴会轨道**：已确认的宴会预定（确定性营收）
- **散客轨道**：AI 预测的散客营收（含置信区间）
- **总预测营收**：两轨合并后的综合预测

#### 模块四：执行计划

- **采购清单**：根据预测需求自动生成，含推荐采购量和告警等级
- **排班方案**：AI 推荐的明日排班，含班次和人数

> 若工作流中的采购/排班阶段已由相关负责人确认锁定，此处显示已锁定版本，右上角标注来源。

#### 模块五：工作流状态

展示备战板关联的 6 阶段工作流（数据收集→分析→采购→排班→审核→确认）的进度和截止时间。

### 4.3 审批操作

1. 核对五个模块数据
2. 如有调整（如修改采购量），在对应模块直接编辑
3. 点击右上角「**一键审批**」按钮
4. 审批后系统自动推送企业微信通知给相关执行人：
   - 采购清单 → 采购负责人
   - 排班表 → 全体员工
   - 特殊营销 → 营销专员

> **快速审批**：若数据无误，可在企业微信推送中直接回复「审批」完成操作。

### 4.4 常见问题

**Q：备战板数据不对，能强制刷新吗？**
A：在备战板页面点击右上角「刷新」图标，或在 URL 末尾加 `?refresh=true`。

**Q：今日备战板未生成怎么办？**
A：检查工作流是否已启动；也可在企业微信发送「生成今日备战板」手动触发。

## 5. 店长功能手册

### 5.1 排班管理

**查看排班**
- 菜单：排班管理 → 周视图 / 日视图
- 支持按员工、岗位、班次筛选

**AI 自动排班（推荐）**

1. 进入「排班管理」→「智能排班」
2. 选择目标日期（建议提前 3 天）
3. 点击「生成排班方案」，AI 根据历史订单预测客流并自动分配
4. 检查结果，如有调整直接拖拽或修改
5. 点击「发布排班」，员工立即收到企业微信通知

**处理换班申请**
- 员工发起换班申请后，店长在「审批管理」页面一键同意/拒绝
- 拒绝需填写原因

**工时统计**
- 「排班管理」→「工时统计」：查看每位员工本周/本月实际工时

### 5.2 库存管理

**日常查看**
- 「库存管理」→「实时监控」：红色=告急，橙色=低库存，绿色=正常
- 过滤「告急」状态，优先处理

**接收告警**
系统自动推送告警（无需手动查看）：
- 库存低于安全线 → 立即推送
- 食材即将过期（7天内）→ 每日提醒
- 食材即将过期（3天内）→ 紧急提醒

**录入盘点数据**
1. 「库存管理」→「盘点」
2. 扫描条码或手动搜索品名
3. 填入实际数量，保存

**查看补货建议**
「库存管理」→「补货计划」：系统根据消耗速度和安全库存自动计算建议采购量。

### 5.3 服务质量

**查看客户反馈**
「服务质量」→「反馈列表」：按来源（App/微信/现场）、情感（正面/负面/中性）、处理状态筛选。

**处理投诉**
1. 点击待处理投诉 → 查看详情和建议处理方案
2. 分配给员工或自行处理
3. 填写处理结果 → 标记完成

> 差评自动旅程：评分 ≤ 2 分且关联会员时，系统自动触发「差评修复旅程」，48小时内完成回访和补偿流程。

**员工服务得分**
「服务质量」→「员工表现」：查看每位员工的服务满意度评分（0-100分），识别需要培训的员工。

### 5.4 食材成本分析

**查看单店差异分析**
「总部看板」→「食材成本差异」→ 选择本店 → 填入日期范围：
- **实际成本率**：库存消耗金额 ÷ 营业收入
- **理论成本率**：BOM 配方标准成本率（加权平均）
- **差异**：正数=超预算，负数=节约
- **Top 10 食材**：消耗成本最高的食材排名

**每日自动告警**（在备战板「昨日复盘」模块）
- 差异 < 2%：正常，无告警
- 差异 2-5%：🟡 偏高，注意
- 差异 ≥ 5% 或实际成本率 ≥ 35%：🔴 严重超标，需排查

**排查方向**
1. 检查 Top 10 食材是否有非正常消耗
2. 核对库存盘点数据是否准确
3. 检查是否有浪费/报损未及时录入

### 5.5 培训管理

**查看员工培训进度**
「培训管理」→「进度追踪」：查看每位员工的课程完成率、考试成绩。

**发起培训任务**
1. 「培训管理」→「培训计划」→「新建计划」
2. 选择培训对象（全员/指定员工/指定岗位）
3. 选择课程模块，设置截止日期
4. 发布后员工收到企业微信通知

**证书管理**
「培训管理」→「证书管理」：查看即将到期的资质证书，系统自动提前 30 天提醒续期。

### 5.6 私域运营

**RFM 用户分层**
「私域运营」→「RFM 分析」：
- S1（高价值）：重点维护，VIP 保鲜旅程
- S2（潜力）：消费激励
- S3（沉睡）：唤醒旅程
- S4（流失预警）：14天内自动触发挽回
- S5（已流失）：低成本召回尝试

**触发旅程**
「私域运营」→「旅程引擎」→「触发旅程」：
选择旅程类型（新客激活/VIP留存/唤醒/差评修复）和目标客户，点击触发。

也可以通过企业微信直接发送：
```
触发新客激活旅程给客户 C001
```

## 6. 总部管理功能手册

### 6.1 跨店看板

**访问**：菜单 → 「总部看板」

展示所有门店的：
- 实时营收排名
- 审批完成率
- 告警门店列表
- 待审批事项汇总

### 6.2 食材成本跨店排名

「总部看板」→「食材成本排名」：
- 按差异率倒序排名（差异最大的门店排第一）
- 快速识别超预算门店（标注 warning / critical）
- 全局摘要：平均成本率、超预算门店数

> 食材成本差异排名每日更新，建议每天晨会前查看。

### 6.3 BOM 配方成本报告

「菜品管理」→「BOM 管理」→ 选择配方 → 「查看成本报告」：
- 每种食材的标准成本明细（含用量、单价、占比）
- 总标准成本和 food cost%
- 食材按成本贡献降序排列，快速定位高成本食材

### 6.4 批量审批

「审批管理」→ 筛选「待审批」→ 批量勾选 → 「批量审批」：
适用于大区管理人员每日处理多店审批需求。

### 6.5 报表导出

各模块均支持导出 Excel/CSV：
1. 进入对应报表页面
2. 点击右上角「导出」按钮
3. 选择日期范围和格式
4. 文件自动下载（大文件通过企业微信发送下载链接）

## 9. 常见问题 Q&A

### 登录相关

**Q：忘记密码怎么办？**
A：登录页点击「忘记密码」→ 输入手机号 → 接收验证码 → 重置密码。如有企业微信账号，建议直接扫码登录，无需密码。

**Q：账号被锁定了怎么办？**
A：连续 5 次密码错误后账号锁定 30 分钟，或联系管理员手动解锁（「用户管理」→ 搜索账号 → 「解锁」）。

---

### 备战板相关

**Q：备战板里的食材成本数据和实际不符？**
A：食材成本使用「库存消耗事务」汇总，如库存盘点录入不及时或有漏录，会导致偏差。请确保每日盘点数据及时录入。

**Q：审批备战板后还能修改吗？**
A：审批后进入「已审批」状态。如需修改，联系管理员将状态回退为「待审批」后重新编辑。

**Q：明日预测营收差异很大，正常吗？**
A：预测基于历史数据和外部因子，遇到特殊活动、节假日等情况误差可能较大。建议参考置信区间范围，而非单一数值。

---

### 排班相关

**Q：AI 排班生成的方案人手不够，怎么处理？**
A：系统会在排班方案中标注「人力缺口」，可在该时段手动添加员工或调整班次，也可向企业微信发送「排班人力缺口」查看具体缺口建议。

**Q：员工排班冲突系统会提醒吗？**
A：会。保存排班时若存在同一员工双重排班，系统自动阻止并提示冲突详情。

---

### 库存相关

**Q：库存告警太多，如何找优先级最高的？**
A：「库存管理」→「实时监控」→ 按「告警等级」排序，**critical（告急）** 优先处理，其次 **warning（低库存）**。

**Q：同一食材多次收到告警，是 Bug 吗？**
A：不是。系统每 6 小时检查一次库存，如问题未解决会持续告警。采购到位后库存数量更新，告警自动消除。

---

### 私域运营相关

**Q：RFM 分层多久更新一次？**
A：每天凌晨自动重新计算。如需立即刷新，可在企业微信发送「重新分析 RFM」。

**Q：旅程触发后多久能看到效果？**
A：新客激活旅程为 7 天 4 触点，完整效果需 7 天后评估。「私域运营」→「旅程引擎」可查看实时执行进度。

---

### 通知相关

**Q：没收到应该收到的企业微信通知？**
A：
1. 确认企业微信账号已关注「智链OS助手」
2. 检查账号绑定是否正确（「账户设置」→「企业微信绑定」）
3. 联系管理员确认通知权限配置

**Q：收到很多重复通知，能减少吗？**
A：「账户设置」→「通知管理」→ 调整各类通知的接收频率或关闭非关键提醒。
