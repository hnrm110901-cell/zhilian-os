## 八、Ontology Adapter兼容性说明
> 与 `apps/api-gateway/src/agents/ontology_adapter.py` (KnowledgeAwareAgent) 的关联

本快照中记录的业务规则，在系统中通过以下路径被Agent使用：
- **KnowledgeAwareAgent.get_dish_bom()**: 查询菜品BOM配方 → 关联本快照"成本管理"规则
- **KnowledgeAwareAgent.get_waste_events()**: 获取损耗事件 → 关联本快照"损耗目标占比"
- **KnowledgeAwareAgent.explain_reasoning()**: 推理证据链 → 关联本快照"因果判断"
- **ontology_sync_service.sync_stores_to_graph()**: PG→Neo4j同步 → 关联本快照"门店清单"
- **waste_reasoning_service**: 损耗五步推理 → 关联本快照"成本管理"规则
