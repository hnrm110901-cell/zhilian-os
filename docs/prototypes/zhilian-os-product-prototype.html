生产配置与运维自动化

prod-health + scheduler-patrol：健康检查与关键任务巡检
prod-monitor-up：Prometheus + Grafana + Alertmanager
prod-alert-e2e：告警注入 -> webhook -> 去重 -> 通知链路校验
