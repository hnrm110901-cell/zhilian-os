

# **《智链OS 宴会管理 Agent PRD 总纲 V1》**





版本：V1.0

产品名称：智链OS Banquet Agent

产品定位：连锁餐饮宴会经营智能体系统

适用业态：大型餐饮 / 海鲜酒楼 / 婚宴酒店 / 连锁宴会餐厅



------





# **一、产品背景**





在中国连锁餐饮行业中，宴会业务是非常重要的收入来源。



典型收入结构：

| **收入类型** | **占比** |
| ------------ | -------- |
| 散客堂食     | 40–60%   |
| 外卖         | 10–20%   |
| 宴会         | 20–40%   |

很多品牌：



> 宴会收入占利润 50%以上



典型业态：



- 海鲜酒楼
- 粤菜
- 湘菜
- 婚宴酒店
- 商务宴请餐厅





------





# **二、行业现状**





当前宴会系统大多属于：

```
宴会预订系统
```

主要功能：



- 客户登记
- 档期管理
- 套餐管理





但缺少：



- 宴会经营分析
- 自动报价
- 宴会资源调度
- 成本管理
- 收入预测





因此很多餐饮企业：



> 宴会管理仍然依赖人工经验。



------





# **三、产品定位**





智链OS 宴会 Agent 是：

```
Banquet Intelligence System
```

系统目标：



让宴会管理从：

```
预订系统
```

升级为：

```
宴会经营系统
```

核心能力：



- 宴会 CRM
- 宴会档期管理
- 宴会套餐管理
- 宴会资源调度
- 宴会执行管理
- 宴会成本与利润分析
- 宴会收入预测

# **五、核心业务流程**

------

- 时间

------





# **7 宴会结算**

- 开票
- 评价
