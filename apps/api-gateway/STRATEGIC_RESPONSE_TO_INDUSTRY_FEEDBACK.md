### 调整2：神经/符号双规机制 - 从"玩具"到"工业级"

**核心问题**：
> "大模型算错了一斤排骨的进价，导致全国1000家店多订了10吨排骨，这个损失谁来赔？"

**解决方案**：Neural-Symbolic Guardrails

```
┌──────────────────────────────────────────┐
│  System 1: 大模型（Neural）               │
│  - 理解意图                               │
│  - 分析趋势                               │
│  - 提出草案                               │
└──────────────────────────────────────────┘
                ↓
┌──────────────────────────────────────────┐
│  System 2: 规则引擎（Symbolic）           │
│  - 硬性校验                               │
│  - 红线拦截                               │
│  - 人类审批                               │
└──────────────────────────────────────────┘
```

**硬性规则示例**：
- ❌ 不可超过历史峰值的120%
- ❌ 不可突破本月资金预算
- ❌ 不可违反食品安全法规
- ❌ 不可超出供应商信用额度

**实施行动**：
1. ✅ 提升decision_validator.py优先级
2. ✅ 增强finance_rule_engine.py

### 调整3：多模态优雅降级 - 语音不是唯一

**核心问题**：
> "后厨轰鸣声90分贝以上，ASR识别率低于95%，员工就会把耳机摘掉不用"

**解决方案**：Graceful Degradation Chain

```
语音交互（Primary）
    ↓ ASR失败 or 噪音>阈值
智能手表震动（Fallback 1）
    ↓ 未响应
前厅POS弹窗（Fallback 2）
    ↓ 未响应
后厨KDS大屏红字（Fallback 3）
    ↓ 未响应
企微/飞书强推送（Final）
```

**关键指标**：
- 连续2次ASR失败 → 立即降级
- 环境噪音>85dB → 自动切换模式
- 紧急指令（催菜、缺货）→ 多通道并发

**实施行动**：
1. ✅ 实现多模态降级链路
2. ✅ 环境噪音实时检测
3. ✅ 关键指令多通道并发
4. ✅ 语音是增强，不是唯一

### 3.2 神经/符号双规机制

**新增核心模块**：
```python
# src/core/neural_symbolic_guardrails.py

class NeuralSymbolicGuardrails:
    """神经/符号双规机制"""

    def validate_ai_decision(self, ai_proposal, context):
        """
        System 1: AI提出草案
        System 2: 规则引擎校验
        """
        # 硬性规则校验
        violations = self.check_hard_rules(ai_proposal)

        if violations:
            # 触发红线，降级为人类审批
            return self.escalate_to_human(ai_proposal, violations)

        return ai_proposal
```
