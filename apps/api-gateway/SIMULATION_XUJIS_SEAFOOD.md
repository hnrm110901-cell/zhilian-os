## 四、日常业务流程

### 4.1 早班流程（7:00-14:00）

**7:00-8:00 开店准备**
- 店长到店，检查门店安全
- 保洁员清洁卫生
- 厨师长检查食材库存
- 采购员根据昨日销售数据采购

**8:00-9:00 食材验收**
- 海鲜供应商送货（每日空运）
- 厨师长验收食材质量
- 配菜员分拣、清洗
- 录入库存系统

**9:00-10:00 备餐准备**
- 厨师准备调料、酱汁
- 配菜员切配
- 服务员摆台、准备餐具
- 收银员检查POS系统

**10:30-14:00 午市营业**
- 10:30 开门迎客
- 11:30-13:00 午餐高峰期
- 服务员点单、传菜
- 厨师出餐
- 收银员结账

**14:00-17:00 午休时段**
- 清洁卫生
- 员工用餐
- 盘点库存
- 准备晚市

### 4.2 晚班流程（14:00-23:00）

**14:00-17:00 准备晚市**
- 晚班员工到岗
- 检查食材库存
- 补充不足食材
- 清洁整理

**17:00-22:00 晚市营业**
- 17:00 开门迎客
- 18:00-20:30 晚餐高峰期
- 服务流程同午市
- 处理客诉、会员服务

**22:00-23:00 收尾工作**
- 清点营业额
- 盘点库存
- 清洁卫生
- 关闭设备
- 店长检查安全

### 4.3 关键业务节点

**采购环节**:
- 每日7:00 海鲜供应商送货
- 每日8:00 蔬菜供应商送货
- 每周一 调料、酒水补货
- 紧急采购：随时

**库存管理**:
- 每日盘点：海鲜（必须当日售完）
- 每周盘点：蔬菜、调料
- 每月盘点：酒水、餐具

**排班管理**:
- 早班：7:00-15:00
- 晚班：14:00-23:00
- 轮休制：每周休息1-2天

**会员管理**:
- 会员充值优惠
- 生日优惠
- 积分兑换
- 专属活动

## 六、智链OS试跑场景

### 6.1 测试场景1：早晨采购与库存管理

**时间**: 7:00-9:00
**涉及模块**:
- Inventory Agent（库存预警）
- 采购系统
- 供应商管理

**测试流程**:
1. 系统根据昨日销售预测今日需求
2. 自动生成采购订单
3. 供应商送货，扫码验收
4. 自动更新库存
5. 异常预警（如海鲜质量问题）

### 6.2 测试场景2：午餐高峰期排班

**时间**: 11:30-13:00
**涉及模块**:
- Schedule Agent（智能排班）
- 客流预测
- 人力调配

**测试流程**:
1. AI预测午餐客流（基于历史数据、天气、节假日）
2. 自动生成排班建议
3. 店长审批
4. 实时监控人效
5. 紧急调配（如突然爆单）

### 6.3 测试场景3：点餐与出餐协同

**时间**: 12:00-12:30
**涉及模块**:
- Order Agent（订单协同）
- 后厨KDS系统
- 语音交互（Shokz耳机）

**测试流程**:
1. 服务员通过语音点单
2. 订单自动传到后厨KDS
3. 厨师按优先级出餐
4. 催菜提醒（语音+KDS红字）
5. 出餐完成通知

### 6.4 测试场景4：客诉处理

**时间**: 12:45
**涉及模块**:
- Service Agent（服务质量）
- SOP知识库
- 客诉处理流程

**测试流程**:
1. 顾客投诉菜品口味不佳
2. 服务员通过语音查询SOP
3. AI推荐处理方案
4. 执行补救措施
5. 记录客诉数据

### 6.5 测试场景5：晚市销售预测

**时间**: 16:00
**涉及模块**:
- Decision Agent（决策支持）
- 销量预测
- 动态定价

**测试流程**:
1. AI预测晚市各菜品销量
2. 建议备货数量
3. 推荐促销菜品
4. 动态调整价格（如临期海鲜）
5. 生成营销方案

### 6.6 测试场景6：闭店盘点

**时间**: 22:30-23:00
**涉及模块**:
- Inventory Agent（库存盘点）
- 财务对账
- 数据分析

**测试流程**:
1. 自动盘点库存
2. 计算损耗率
3. 对账营业额
4. 生成日报
5. 异常预警（如损耗过高）
