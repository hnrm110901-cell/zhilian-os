### 第8页：智能决策引擎
- **决策优先级公式**
```
得分 = 0.40×财务影响 + 0.30×紧急程度 + 0.20×置信度 + 0.10×执行难度
```
- **4个决策窗口**
  - 08:00 晨推 → 今日采购建议
  - 12:00 午推 → 午市实时调整
  - 17:30 战前 → 晚市备货/排班
  - 20:30 晚推 → 当日复盘/次日预警
- **决策样例**
  - "建议减少明日虾采购量20%（库存充足），预期节省¥340，置信度82%"
  - [一键确认] [查看详情] [忽略]

## 三、成本节省测算模型

### 模型假设
```
门店：徐记海鲜五一广场店 (XJ-CS-001)
月营收：¥180万（日均¥6万）
当前食材成本率：35%
目标食材成本率：33%（降2个点）
当前人工成本率：22%
目标人工成本率：20%（降2个点）
```

### 计算明细

#### A. 食材成本优化
```
月食材成本（当前）：¥1,800,000 × 35% = ¥630,000
月食材成本（目标）：¥1,800,000 × 33% = ¥594,000
月节省：¥36,000
年节省：¥432,000

优化路径：
1. BOM标准化：消除配比不一致 → 降0.5个点（¥108,000/年）
2. 损耗管控：Top5食材损耗降15% → 降0.8个点（¥172,800/年）
3. 采购优化：需求预测精准采购 → 降0.4个点（¥86,400/年）
4. 价格监控：供应商比价 → 降0.3个点（¥64,800/年）
合计：2.0个点（¥432,000/年）
```

#### B. 人力成本优化
```
月人工成本（当前）：¥1,800,000 × 22% = ¥396,000
月人工成本（目标）：¥1,800,000 × 20% = ¥360,000
月节省：¥36,000
年节省：¥432,000

优化路径：
1. 智能排班：客流预测匹配 → 减少冗余工时 → ¥172,800/年
2. 峰谷调度：高峰加人/低谷减人 → ¥129,600/年
3. 技能匹配：合适的人在合适的岗 → ¥129,600/年
```

#### C. 运营效率提升
```
采购效率提升30%：节省采购员工时 → ¥24,000/年
决策响应加速：从"事后"到"事中" → 减少损失 → ¥60,000/年
```
